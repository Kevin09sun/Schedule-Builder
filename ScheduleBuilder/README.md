# Schedule Builder

ICS4U1 project by Kevin Sun.

## Requirements

- Java 8 or later (`javac` and `java` on your PATH)

## Clone, Compile, and Run

```bash
git clone https://github.com/Kevin09sun/Schedule-Builder.git
cd Schedule-Builder
javac ScheduleBuilder/src/Main.java ScheduleBuilder/src/data/*.java ScheduleBuilder/src/logic/*.java ScheduleBuilder/src/model/*.java
java ScheduleBuilder.src.Main
```

> Make sure to run the java command from the root of the repository containing ScheduleBuilder/ folder. The csv files are relative to that location
